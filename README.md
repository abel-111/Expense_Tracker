# Expense Tracker

A web application built with Python and Flask to track personal expenses.

## Current Features

- Add expenses with category and amount
- View all expenses in a list
- Edit existing expenses
- Delete expenses
- View total spending
- Add expenses with category and amount and date
- View all expenses with date in the list
- Edit existing expenses including date
- Delete expenses
- View total spending

## Technologies Used

- Python 3.14
- Flask 3.1.3
- SQLite (built-in Python library)
- Jinja2 (Flask templating)
- HTML

## Project Structure

expense-tracker/
├── app.py
├── expenses.db
└── templates/
    ├── home.html
    └── edit.html

## How to Run

1. Clone the repository
   git clone <your-repo-url>

2. Install Flask
   pip install flask

3. Run the app
   python app.py

4. Open browser and visit
   http://127.0.0.1:5000
